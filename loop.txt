## 0601- Write a program to print all natural numbers from 1 to 10 using loop.(using while loop,for loop)
void main() 
{
  for(int i=1;i<=10;i++)
  {
    print(i);
  }
}

void main()
 {
  int i = 1;
  while (i <= 10) 
{
    print(i);
    i++;
  }
}
